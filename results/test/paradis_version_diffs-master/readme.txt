This folder contains the output of diff between the source code files of:
	1) the default (v2.5.1) and the Grand Challenge ParaDiS (default_gc/default_gc_*filename*)
	2) the default (v2.5.1) and the HPC optimized ParaDiS (default_optimized/default_optimized_*filename*)


Additionally:
File "input_output.txt" is the summary of differences in the required input files and the program output between the different versions.
File "ParaDiSUsersGuide-V2.5.1-Pub.pdf" is the user guide of the default ParaDiS
File "compiling_running.txt" contains information about the required modules and run parameters
Folder "test_run" holds a test case for ParaDiS with precipitate implementation with the shell script and input files. 

For more information and citing:
Lehtinen, A., Granberg, F., Laurson, L., Nordlund, K., & Alava, M. J. (2016). Multiscale modeling of dislocation-precipitate interactions in Fe: From molecular dynamics to discrete dislocations. Physical Review E, 93(1), 013309.
